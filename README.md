# ⏱️ Reaction Timer

A simple and interactive **Reaction Timer Game** that measures how quickly a user reacts to a visual signal.

The application introduces a random delay before showing the signal and records the user's reaction time once they click. This project demonstrates **JavaScript timing logic, event handling, and DOM manipulation**.

---

## 🚀 Features

- Random delay before the signal appears
- Measures reaction time in **milliseconds**
- Prevents early clicks
- Displays reaction time instantly
- Simple and responsive UI

## 🎮 How It Works

1. Click **Start**
2. Wait for the screen to change color
3. Click immediately once the signal appears
4. The application calculates and displays your **reaction time**

If the user clicks too early, the game resets and prompts them to try again.
